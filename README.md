# Projeto Playwright — Preenchimento Automático de Formulário

Este projeto demonstra como utilizar Playwright para preencher automaticamente um formulário público e validar a submissão.

## Tecnologias
- Node.js
- Playwright
- TypeScript

## Instalação
```bash
npm install
npx playwright install
```

## Como Rodar os Testes
```bash
npm test
```
